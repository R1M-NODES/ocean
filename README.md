<b> Install Ocean </b>

```
bash <(curl -s https://raw.githubusercontent.com/R1M-NODES/ocean/master/install.sh)
```

Deleta Ocean 

```
cd ocean && \
docker-compose down
```

```
cd && \ rm -rf ocean
```

```
rm -rf ocean
```


Restart Ocean 

```
cd ocean && \
docker-compose restart
```
